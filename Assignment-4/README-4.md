# STEVE'S CHAT // EST. 2001
**CampusPe Gen AI Assignment — Chat UI Development**

A Y2K skate zine inspired chat interface built with HTML, CSS, JavaScript, jQuery and Bootstrap 5.
Inspired by Thrasher Magazine circa 2001.

---

## How to Run

1. Unzip the folder
2. Open `index.html` in any modern browser
3. No install or server needed — runs entirely in the browser

## Design

Y2K skate zine aesthetic inspired by Thrasher Magazine 2001.

- **Misty Blue** `#C4D2DA` — base background
- **Poinciana** `#D94A4A` — all text, buttons, accents
- **Scallop Shell** `#F3D6C6` — borders and details
- **Bebas Neue** — display font for that zine feel
- **Courier Prime** — monospace body font

---

## Technologies

- HTML5 — semantic structure
- CSS3 — custom properties, flexbox, animations
- Bootstrap 5.3 — responsive grid and utilities
- Font Awesome 6 — icons
- Google Fonts — Bebas Neue and Courier Prime
- jQuery 3.7 — DOM manipulation and events
